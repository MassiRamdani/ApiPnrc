## Run locally

To build and run this application locally, you'll need latest versions of Git. From your command line:

```
# Clone this repository
$ git clone https://github.com/MassiRamdani/ApiPnrc.git

# Go into the repository
$ cd ApiPnrc

# Run
$ node server.js

```
